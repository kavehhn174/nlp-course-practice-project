This practice project is intended for the final project presentation of Natural Language and Speech course in my undergraduate studies (Fall 2023).
The project was initially done at Dec 2023, However, it was uploaded to GitHub at Jan 2025
The project includes Preprocessing, Spell Correction and Topic Classification and most of the works are done manually rather than using libraries.

Used Packages: NLTK, Numpy, PyEnchant
